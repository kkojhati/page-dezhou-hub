# page-dezhou-hub

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库中的页面彼此独立，不针对任何特定域名或网站，主要用于保存和展示静态 HTML 内容。

## 项目简介

本仓库收集了若干独立的 HTML 页面，每个页面均可单独运行或查看。这些页面可能包含信息展示、工具界面、实验性设计等，适合作为静态资源存档或轻量级页面集合。

## 目录说明

```
├── pages/            # 存放所有 HTML 页面文件
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/           # 页面引用的公共资源（CSS、JS、图片等）
├── index.html        # 仓库首页或导航页（如有）
└── README.md         # 本文件
```

- `pages/`：每个文件为一个独立页面，文件名即页面标识。
- `assets/`：存放页面可能共享的样式、脚本或媒体文件。
- `index.html`：可选，用于提供页面列表或跳转入口。

## 页面归档说明

- 所有页面均为纯 HTML 实现，部分可能包含内联 CSS/JS。
- 页面之间无依赖关系，可单独访问或部署。
- 归档内容以稳定、可长期访问为原则，不保证时效性或功能性更新。
